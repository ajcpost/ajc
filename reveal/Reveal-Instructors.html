Reveal.js for Instructors Edit

Cross-Platform HTML Slideshows

Reveal.js is an open source framework for creating amazing interactive slideshows using HTML. The presentations you create look great and function on a variety of devices and screen sizes (no proprietary software required) and are easy to upload to the web or place in your D2L course. In this short tutorial, we'll explore the basics of getting started with the framework. Once you're finished, you may want to visit the following resources for additional information:

  • An example reveal.js slideshow
  • Reveal.js project homepage
  • Online Presentations with reveal.js (Lynda.com course)
  • Tutorial-Reveal.js for Beginners (step-by-step tutorial for novices)
  • Visual editor for creating slideshows with reveal.js

This tutorial assumes that you have a basic understanding of HTML and CSS. Even if you don't, you should be able to follow along, but you may have a difficult time creating your own modifications in the future.

Setting up your slideshow

To start, we need to download the project to your computer. Head over to  https://github.com/hakimel/reveal.js and click on the Download Zip button in the right column of the project page. Once the download is complete, go ahead and unzip the package. You should now have a folder containing the items shown in the screenshot below.

Now that we've downloaded all the necessary files, let's create our first project. Start by creating a new folder on your computer. Feel free to name it whatever you like; I'm going to call mine "Reveal Example." Then, copy the following files from the reveal.js-master folder you previously unzipped.

  • css
  • js
  • plugin
  • lib

In reveal.js, each slideshow is contained in a single HTML file. To create a new slideshow, let's add an HTML file to our folder (I'm calling mine Example 1). Here's what you should see now:

Before we can start adding content to our slideshow, we need to copy and paste the following boilerplate code into our new, empty HTML file. 

<!doctype html>
<html lang="en">    
<head>
    <meta charset="utf-8">
    <title>Reveal.js 3 Slide Demo</title>
    <link rel="stylesheet" href="css/reveal.min.css">
    <link rel="stylesheet" href="css/theme/default.css" id="theme"> 
    <!--Add support for earlier versions of Internet Explorer -->
    <!--[if lt IE 9]>
    <script src="lib/js/html5shiv.js"></script>
    <![endif]-->
</head>

<body>
    <!-- Wrap the entire slide show in a div using the "reveal" class. -->
    <div class="reveal">
        <!-- Wrap all slides in a single "slides" class -->
        <div class="slides">

            <!-- ALL SLIDES GO HERE -->
            <!-- Each section element contains an individual slide -->
            <section>
            <!-- The content of your slide goes here --!>
            </section>



        </div>
    </div>
    <script src="lib/js/head.min.js"></script>
    <script src="js/reveal.min.js"></script>

    <script>
        // Required, even if empty.
        Reveal.initialize({
        });
    </script>
</body>
</html>

It's not really important for you to know what all that code does at this point. The essential concepts are:

  • Each slide is wrapped in a <section> div. You can put anything in the section you'd like - headings, text, images, embedded videos, etc.
  • All of the slides are wrapped in a <div> with a class of reveal - <div class="reveal"></div>

That's pretty much it! You can add as many slides as you'd like by creating more <section> tags in your HTML document. Now that you know the basics of creating a slideshow and adding individual slides, let's look at some other cool features of reveal.js.

Changing your theme

You can easily change your theme by modifying a single line of code in your header. Notice the <link rel="stylesheet" href="css/theme/default.css" id="theme"> reference in the boilerplate code we copied in earlier. Change the default.css reference to any other CSS file in the theme folder. reveal includes several by default, but you can also create your own or modify one of the default themes if you'd like! In this example, I've chosen the Sky theme - <link rel="stylesheet" href="css/theme/sky.css" id="theme">

Linking to other slides

You can easily link from one slide to another if you provide an ID when creating the section. For example, if you have a slide setup like this:

<section id="second-slide">
    <h2>Second Slide</h2>
    <p>Example Test</p>
</section>

You could link to it from any other slide like this:

<p>Take me to slide<a href="#/second-slide"> 2</a>

Nested Slides

Normally, slides are arranged in a linear order. You move from left to right through the slide deck using the arrow keys on your keyboard. However, you can also create nested slides, which allow you show additional information about a topic by moving Up and Down through a series of vertical slides. This is great for times when you need to dig deeper into a topic. To create nested slides, just place additional <section></section> tags inside the main <section> div for the current slide as shown in the example below.

<section>
<h2>Main Slide</h2>
    <section>
        <h3>Nested Slide 1</h3>
    </section>
    <section>
        <h3>Nested Slide 2</h3>
    </section>
    <section>
        <h3>Nested Slide 3</h3>
    </section>
</section>

Fragmented Views

Normally, all of the content for a slide will be visible as soon as you navigate to it. However, you can load items separately by adding a class="fragment" to the element. In the example below, we are loading each item in the bulleted list separately.

<section>
    <h2>loading lists items individually</h2>
    <ul>
        <li class="fragment">List Item 1</li>
        <li class="fragment">List Item 2</li>
        <li class="fragment">List Item 3</li>
    </ul>
</section>

Presenting with reveal.js

Navigating through a completed slideshow is very intuitive. Start off by opening the HTML page in a modern browser of your choice. Once opened, you can use your keyboard's space key or arrow keys to move through the presentation. If you're using a touch-enabled device, you can also swipe your way between slides. Here are some other handy shortcuts to use while presenting:

  • Pressing the Escape key shows a thumbnail view of all slides. You can navigate to a specific slide by clicking on the thumbnail.
  • Press the . key to darken the screen.
  • If you're using a mouse, you can also click the arrow buttons in the bottom right corner of the slide to move between slides.
