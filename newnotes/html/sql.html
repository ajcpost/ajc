WHERE vs JOIN (Implicit vs Explicit joins)

 
SELECT * 
FROM product 
LEFT JOIN sales on sales.product=product.id 
WHERE product.id=2

* This will return Product 2 and sales transactions for this product.
* Doing explain "query" will show only one row is fetched from product table

 
SELECT * 
FROM product 
LEFT JOIN sales on sales.product=product.id and product.id=2

* This will return all Products but only Product "2" will have associated 
  sales transaction. All others will have NULL output.
* Doing explain "query" will show all row are fetched from product table

For INNER JOIN, these mean the same thing but for OUTER JOINS it will give different reasons.
