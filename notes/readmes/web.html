    • Set JRE_HOME (/System/Library/Frameworks/JavaVM.framework/Versions/1.5.0/Home)
    • Deploy the web application in install_dir/webapps
    • install_dir/bin/shutdown* install_dir/bin/start*
  • Logs
    • check for errors in install_dir/logs/cat*<date> during startup
