Mysql

Deployment

  • Typical location /usr/local/mysql
  • Auto-start variable: /etc/hostconfig MYSQLCOM=-YES-
  • To reinitialize if not able to start mysql (all DBs will be deleted!)
    • startup errors seen : Can't find file: './mysql/host.frm
    • rm -rf install_dir/data
    • sudo scripts/mysql_install_db --user=mysql (Important to use mysql as a user so as to have correct permissions)
    • check passwords: bin/mysql -u root, SELECT User, Host, Password FROM mysql.user;
    • set passwords: bin/mysqladmin -u root password root

Basic usage

  • To start
    • if auto-start "YES": "sudo /Library/StartupItems/MySQLCOM/MySQLCOM start"
    • else: sudo su, cd install_dir ./bin/mysqld_safe --user=mysql &
  • bin/mysql -u root -p, password is "mysql"
  • Use <database>;
  • show tables;
  • System database
    • use mysql;
    • select user, host from user;
